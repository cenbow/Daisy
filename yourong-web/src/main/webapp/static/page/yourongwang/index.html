什么是人气值

人气值是有融网向用户提供的一种可兑换成现金券的奖励，
兑换后可直接用于投资。

人气值的用途

1、人气值可兑换现金券，1点人气值等于1元现金券，
      现金券可用作投资；

2、人气值可用于报名参加有融网部分活动。

如何获得人气值

1、每日签到可随机获得1~6点人气值;

2、邀请好友注册并投资(单笔投资额≥￥500)，
      推荐人即可获得好友投资额2‰的人气值；

3、用户可参与有融网推出的活动，赢取人气值。
      例如：一羊领头：28点人气值    查看详情>>

温馨提示: 1、邀请好友注册时，
      须好友输入推荐人的手机号码； 2、好友通过推荐人唯一的邀请链接
      进入注册页面成功注册。

如何兑换人气值

Step1：用户登录有融网，进入“我的账户—我的优惠—我的人气值”；
Step2：人气值集满50、100、500、1000即可点击领取现金券的按钮，兑换相应面额现金券。
注：人气值不满50，则无法进行兑换。

人气值的使用规则

1、人气值可累计，无上限，永久有效；
2、人气值不可转让、不可折现。
