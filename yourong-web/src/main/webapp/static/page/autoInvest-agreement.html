有融网自动投标协议书

第1条：浙江小融网络科技股份有限公司（简称“小融网络”）依据本协议的规定在有融网网站 （由小融网络负责运营的提供借贷交易居间服务和债权项目信息服务的互联网平台，本协议中有融网与小融网络视为同一主体, 以下统称“有融网”；网址：www.yrw.com）上为有融网会员（简称“投资人”）提供自动投标服务。本协议在投资人和小融网络间具有法律效力。

第2条：有融网建议您认真阅读本协议，并在充分理解各条款内容后再选择是否接受本协议。除非您接受本协议所有条款， 否则无权使用有融网于本协议项下所提供的自动投标服务。您一经开启自动投标服务即视为完全接受本协议全部条款。

第3条：本协议中的“投标”即为“出借”或“受让”之意。

第4条：自动投标是有融网为了方便投资人而提供的一种智能投标委托代理服务。

第5条：自动投标的对象为有融网平台上除新手项目外所有处于可投状态的投资理财项目。投资人启用自动投标服务，即视为同意并接受有融网所有投资理财项目的投资流程、规则及其他相关服务协议。

第6条：启用自动投标服务，投资人可以根据有融网自动投标规则设定投标方案。小融网络会根据投资人设定的方案筛选合适的项目标的进行自动投标。

第7条：设定投标方案后，即视为投资人在此授权小融网络代其通过有融网的自动投标服务将投资人的账户余额进行自动投标。 有融网的投标行为等同于投资人的自行投标，投资人不得以任何理由要求撤销该等投标行为，交易中产生的任何风险和损失由投资人自行承担。

第8条：投资人应知悉自动投标服务无法保证不错过任何满足其定制条件的标的。由此产生的损失，有融网不承担任何责任。

第9条：自动投标发生后，有融网会向投资人绑定的邮箱或手机发送通知。通知从有融网发出时即视为已送达，投资人不会以任何理由否认该笔投资。

第10条：有融网自动投标具体规则，以网站最新公布为准。

第11条：有融网网站自动投标功能由投资人自愿决定开启和关闭，投资人关闭自动投标功能后即视为终止授权有融网自动投标。

第12条：小融网络拥有对本协议的修改权及最终解释权。
