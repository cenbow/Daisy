项目介绍
本项目借款主要用于借款人资金周转。借款人为桐乡市豪**汽车销售有限公司，主要经营中高档机动车和二手车的销售、机动车零配件的销售等业务，注册资金3000万元整，信用良好，该公司的法定代表人资产丰厚；借款人的还款来源为公司经营收入，有充足的偿还能力，而且该公司以其正在销售的东风日产奇骏(2.5L)CVT领先版汽车的汽车合格证作质押，该汽车厂家指导价为人民币22.88万元整，出借额度为人民币15万元整，同时该公司法定代表人承诺若借款人逾期还款的，法定代表人自愿以全部资产承担连带责任。另外，出借人经营一家电子商务公司，并且在杭州有固定资产，资产丰厚，出借人对项目债权承诺逾期赎回。（若质押期间客户有意愿购买该车辆，借款人将用不低于该车辆价值的其他车辆对应的汽车合格证置换）

保障措施

借款撮合及债权推荐

借款项目由杭州小融投资咨询管理有限公司推荐，该公司保证债权真实有效。

风控师审核

质押汽车合格证所对应的东风日产奇骏(2.5L)CVT领先版汽车厂家指导价人民币22.88万元，借款人民币15万元整，严格控制了借款金额，风控师审核适批。

质押管理

项目借款人提供的质押汽车合格证由项目推荐方或其委托的第三方存放于指定的档案室中，严密看管，并定期或不定期委派人员至借款单位检查车辆存放及销售情况。

法律监管

专业律师对平台模式及债权形成流程进行严格法律风险把控，并提供法律保障。

资金监管

投资资金由新浪支付进行资金托管，实行专人专户、专款专用实现实时监管，资金零挪用。
